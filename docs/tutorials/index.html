Tutorials

Six builds you can ship.

Each tutorial is a complete walkthrough with prereqs, step-by-step instructions, runnable code, and a reference repo. Pick by what you're building, not by what's flashiest.

01$87 $2.50

Cross-border payment widget

Drop-in React component. User picks recipient + amount in their currency, widget quotes + executes the cheapest source asset. Replaces $80+ SWIFT wires with $2 swaps.

React · sera.pay_invoice25 min
02read-only

FX trading dashboard

Live FX dashboard reading Sera's reference rates + multi-source mid + executable depth. Spot deviations across 60+ stablecoin FX pairs. No wallet required.

Next.js · find_deals30 min
03on-chain

Prediction market on FX

Binary markets like "USD/JPY > 158 by Friday" using Sera's reference rate as the settlement oracle. Payouts route through Sera FX so winners get paid in any currency.

Solidity · get_fx_rate60 min
04cron

Treasury rebalancer

Cron job that reads multi-currency wallet balances, computes drift, emits Sera swap intents. Runs daily, signs nothing.

Node · rebalance_plan20 min

x402-paid API

Charge AI agents per call in USDC over plain HTTP. Standard 402 → pay → 200. No paywall pages, no Stripe.
