# Code of Conduct

## The short version

Be decent. Don't be a jerk. Help people who are clearly trying.

## The longer version

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). Standard expectations: be respectful, assume good faith, give constructive feedback, don't harass anyone, don't post anything hateful or discriminatory.

## Enforcement

Issues with conduct can be reported via [GitHub's private security advisory](https://github.com/Josh-sera/sera-mcp/security/advisories/new) or by emailing the maintainer (see GitHub profile). All reports will be reviewed and acted on; the maintainer's decision is final.

Maintainers may remove comments, lock issues/PRs, or ban contributors for behavior that violates this code.

## Scope

Applies to all project spaces — issues, PRs, discussions, social accounts associated with the project — and to anyone representing the project in public.
